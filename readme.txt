== Description ==

**Oasiscatalog - Product Importer**

A professional solution for integrating your WordPress store with the OasisCatalog wholesale catalog. The plugin automatically imports products, prices, inventory, and descriptions, saving you time on manually populating your website.

**Requirements:**
- Active WordPress with WooCommerce
- API key and UserID from your [OasisCatalog account](https://www.oasiscatalog.com/cabinet/integrations)

== External services ==

This plugin connects to the API to retrieve data from the Oasis catalog. Product attributes, descriptions, brands, and logo application methods are included.

The plugin can also transfer information about orders from your website that include OasisCatalog products to your OasisCatalog personal account.

To connect to the API, you need a key from your [personal account](https://www.oasiscatalog.com/cabinet/integrations)
